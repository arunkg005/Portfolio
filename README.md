# Arun Kumar Gupta — Software Developer & ML Enthusiast

A premium, fully responsive single-page portfolio website featuring a modern **dark mode neumorphic & glassmorphic design**. 

Live Demo / Preview available by opening `index.html` in any modern web browser.

## ✨ Features

- **Premium Dark Neumorphic Aesthetic**: Vibrant, curated color palette with custom HSL tailored colors and smooth neumorphic shadows/depth.
- **Dynamic Projects Section**: Interactive project cards displaying software development and machine learning projects with image overlays and links.
- **Interactive Certifications**: A dedicated section showcasing earned and in-progress certifications (ServiceNow, Salesforce, etc.) with custom status badges and click-to-expand details using spring-like modal animations.
- **Responsive Layout**: Fluid design that automatically adapts to desktop, tablet, and mobile screens.
- **Contact Form**: A clean contact form with dynamic error handling and feedback states.

## 🛠️ Built With

- **HTML5**: Semantic, accessible markup.
- **CSS3 (Vanilla)**: Maximum performance, custom properties (CSS variables), grid/flexbox layout, and custom micro-animations.
- **JavaScript (ES6+)**: Custom animations, dynamic modal state management, and contact form handling.

## 📁 Repository Structure

- `index.html` — Core markup and styling.
- `assets/` — Project images and media.
- `.gitignore` — Standard ignore patterns.
